    method

    addAnalogDataItem

    AnalogDataItem DataItems that represent continuously-variable physical quantities ( e.g., length, temperature), incontrast to the digital representation of data in discrete items NOTE Typical examples are the values provided by temperature sensors or pressure sensors. OPC UA defines a specific UAVariableType to identify an AnalogItem. Properties describe the possible ranges of AnalogItems.

    example:

    namespace.add_analog_dataItem({ componentOf: parentObject, browseName: "TemperatureSensor",

     definition: "(tempA -25) + tempB",
 valuePrecision: 0.5,
 //-
 instrumentRange: { low: 100 , high: 200}, // optional
 engineeringUnitsRange: { low: 100 , high: 200}, // mandatory
 engineeringUnits: standardUnits.degree_celsius,, // optional

 // access level
 accessLevel: 1
 minimumSamplingInterval: 10,

    });

    Parameters

    • options: AddAnalogDataItemOptions

    Returns UAAnalogItem
